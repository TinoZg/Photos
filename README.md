# Photos

## Kako koristiti:

U liniji koda 12 **photosFolderPath** napisati putanju do foldera gdje se nalaze sve slike. Ne smije biti pod foldera. Nakon toga će program u istom folderu napraviti strukturu podfoldera i slike prebaciti u odgovorajuće foldere na temelju godine i mjeseca kada su slikane. Program ne mijenja naziv slika i trenutno radi za format JPG.
